# Портреты рас

## Структура файлов

### ⭐ Новая организация (для редактора персонажа)

```
races/
├── {raceId}/           # Папка для каждой расы (dwarf, human, elf, и т.д.)
│   ├── m/              # Мужские портреты
│   │   ├── base.png    # Базовый портрет (обязательный)
│   │   ├── w.png       # Портрет подрасы "Война" (War)
│   │   ├── k.png       # Портрет подрасы "Знание" (Knowledge)
│   │   ├── c.png       # Портрет подрасы "Общество" (Community)
│   │   ├── s.png       # Портрет подрасы "Тень" (Shadow)
│   │   ├── m.png       # Портрет подрасы "Мистика" (Mysticism)
│   │   └── n.png       # Портрет подрасы "Природа" (Nature)
│   └── f/              # Женские портреты
│       ├── base.png    # Базовый портрет (обязательный)
│       ├── w.png       # War
│       ├── k.png       # Knowledge
│       ├── c.png       # Community
│       ├── s.png       # Shadow
│       ├── m.png       # Mysticism
│       └── n.png       # Nature
```

**Размер**: 512x512px или выше  
**Формат**: PNG с прозрачностью

#### Логика загрузки в редакторе персонажа:

1. Если выбрана подраса (аспект): `/races/{raceId}/{gender}/{shortCode}.png` где shortCode:
   - `w` = War (Война)
   - `k` = Knowledge (Знание)
   - `c` = Community (Общество)
   - `s` = Shadow (Тень)
   - `m` = Mysticism (Мистика)
   - `n` = Nature (Природа)
2. Если подраса не выбрана или файл не найден: `/races/{raceId}/{gender}/base.png`
3. Fallback на старую структуру: `/races/portraits/{raceId}_{gender}.png`

#### Список рас (12 штук):
dwarf, human, tiefling, drow, goblin, halfelf, halfling, halforc, dragonborn, elf, gith, gnome

#### Коды аспектов (6 букв):
w, k, c, s, m, n (War, Knowledge, Community, Shadow, Mysticism, Nature)

**Всего файлов**: 2 пола × 12 рас × 7 вариантов = **168 портретов**

### Иконки для графа (круглые миниатюры)
Расположение: `/public/images/races/icons/`  
Формат: PNG с прозрачностью или JPG  
Размер: 120x120px (отображаются в круге ~80-100px)  
Именование: `{race-id}-icon.png` или `.jpg`

Примеры:
- `dwarf-icon.png`
- `human-icon.png`
- `tiefling-icon.png`
- и т.д.

### Полные портреты для описания (старая структура)
Расположение: `/public/images/races/portraits/`  
Формат: PNG или JPG  
Размер: 400x500px (портретная ориентация)  
Именование: `{race-id}-portrait.png` или `{race-id}_{gender}.png`

Примеры:
- `dwarf_m.png`, `dwarf_f.png`
- `human_m.png`, `human_f.png`
- и т.д.

## Рекомендации по изображениям

### Портреты персонажей (новая структура):
- Портрет лица или верхняя часть тела
- Чёткие расовые черты + характер аспекта
- Нейтральный фон или тематический (война, знание, природа и т.д.)
- Детализация для масштабирования (zoom до 150%)

### Иконки для графа:
- Крупный план лица
- Хорошо узнаваемые черты расы
- Контрастность для читаемости в маленьком размере

### Портреты для описания:
- Портрет в полный рост или 3/4
- Демонстрирует стиль и особенности расы
- Качественная детализация
- Атмосферное освещение

## Fallback
- Редактор персонажа: `/races/{raceId}/{gender}/base.png` → `/races/portraits/{raceId}_{gender}.png`
- Граф рас: эмодзи-иконка из races.json
