# Изображения полов для редактора персонажа

## Структура файлов

```
gender/
├── male.png      # Силуэт мужского персонажа
└── female.png    # Силуэт женского персонажа
```

## Требования к изображениям

**Размер**: рекомендуется 120×200px или пропорционально (соотношение ~3:5)
**Формат**: PNG с прозрачностью
**Стиль**: Силуэты, четкие контуры, узнаваемые фигуры

## Использование

Изображения отображаются на первом шаге редактора персонажа (выбор пола).

### Позиционирование в коде:
- Male: центрирован в точке `(canvasSize/2 - 100, canvasSize/2)`
- Female: центрирован в точке `(canvasSize/2 + 20, canvasSize/2)`
- Изображение: `x="-60", y="-100"` (относительно точки позиционирования)
- Размер: `width="120", height="200"`

### Анимация:
При выборе пола:
- Male: улетает влево (`translateX(-300px)`)
- Female: улетает вправо (`translateX(300px)`)
- Длительность: 400ms ease-in

### Fallback:
Если изображение не найдено, отображается:
- Прямоугольник с символом ♂ или ♀
- Цвета: синий (#3b82f6) для мужского, розовый (#ec4899) для женского
