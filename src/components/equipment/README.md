# Equipment Components / Экипировка

Компоненты для выбора и управления экипировкой персонажа.

## При создании персонажа

### EquipmentSelector.vue
Выбор начальной экипировки на этапе создания.

**Функционал:**
- Выбор брони по слотам
- Выбор оружия с учётом эпохи и богатства
- Фильтрация по типу и характеристикам
- Интеграция с ArmorCarousel и WeaponSelector

**Данные:**
- `items.json` - база предметов
- `itemTypes.json` - типы предметов
- `epochs.json` - эпохи (ограничения оружия)
- `wealths.json` - уровни богатства

### ArmorCarousel.vue
Карусель выбора брони.
- Горизонтальный скролл
- Категории: голова, торс, руки, ноги
- Характеристики защиты

### WeaponSelector.vue
Выбор оружия с расширенными фильтрами.
- Категории: ближний бой, дальний бой
- Фильтры по типу и характеристикам
- Ограничения эпохи

## В игре

### EquipmentManager.vue
Управление экипировкой во время игры.

**Функционал:**
- Смена активного оружия
- Наборы оружия (weapon sets)
- Характеристики защиты
- Быстрый доступ

### InventoryPanel.vue
Инвентарь персонажа.

**Функционал:**
- Список предметов
- Использование предметов
- Передача/выброс
- Категории и сортировка

{

  type: 'weapon',
  subtype: 'sword',
