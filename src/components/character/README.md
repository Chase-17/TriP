# Character Components / Персонажи

Компоненты для создания, отображения и редактирования персонажей.

## Создание персонажа

### CharacterWizard.vue
Обёртка визарда создания персонажа.
- Интеграция с stores
- Обработка constraints от мастера
- Финализация и сохранение персонажа

### CharacterCreationCanvas.vue (~1900 строк)
Многошаговый визард создания персонажа.

**Шаги:**
1. Раса и подраса
2. Пол и возраст
3. Класс
4. Характеристики и навыки
5. Экипировка
6. Финализация

**Использует:**
- `characterCreationStore` - состояние создания
- Данные: `races.json`, `subraces.json`, `classes.json`, `aspects.json`

### CharacterFinalization.vue
Финальный шаг создания:
- Ввод имени
- Выбор портрета (preset или AI-сгенерированный)
- Предпросмотр и подтверждение

### ClassStatsAndSkills.vue
Выбор характеристик и навыков на основе класса.

### AspectChart.vue
Радарная диаграмма аспектов персонажа (Canvas).

## Отображение персонажа

### MobileCharacterSheet.vue (~3600 строк)
Полноценный лист персонажа для мобильных устройств.

**Табы:**
- Статы и информация
- Экипировка
- Инвентарь
- Навыки и способности

**Взаимодействие:**
- Смена наборов оружия
- Использование предметов
- Броски характеристик

### CharacterPortrait.vue
Компактный портрет персонажа.
- Fallback аватар
- Рамка по классу
- Размеры: sm/md/lg

## Stores
- `characterCreationStore` - процесс создания
- `charactersStore` - все персонажи игры
