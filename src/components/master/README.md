# Master Components / Инструменты Мастера

Компоненты доступные только в режиме мастера (isMaster=true).

## MasterTools.vue
Панель инструментов мастера.

**Режимы:**
- Просмотр (view)
- Перемещение токенов (move)
- Рисование terrain (paint)
- Выбор (select)

**Настройки:**
- Показ сетки
- Показ направлений
- Настройки отображения

## MasterSceneTools.vue (~2000 строк)
Управление игровой сценой.

**Функционал:**
- Управление ходами (turn order)
- Броски от имени NPC/монстров
- Групповые броски
- Условия и эффекты
- Публикация в лог

**Взаимодействия:**
- Выбор целей атаки
- Применение урона
- Состояния персонажей

## MasterCharactersPanel.vue
Управление персонажами.

**Табы:**
- Все персонажи
- Только игроки
- Только NPC

**Функционал:**
- Быстрое изменение HP
- Применение ран
- Синхронизация с игроками
- Создание NPC на лету

## NpcPanel.vue
Создание и редактирование NPC/монстров.

**Функционал:**
- Шаблоны NPC из `npcTemplates.json`
- Кастомные NPC
- Быстрое добавление на карту
- Редактирование характеристик

## SceneTemplatesList.vue
Шаблоны быстрого запуска сцен.

**Шаблоны включают:**
- Предустановленные terrain
- Начальные позиции токенов
- Условия освещения
- Описание сцены

## Stores для мастера
- `battleMapStore` - управление картой
- `sceneLogStore` - публикация событий
- `sessionStore` - P2P соединения
