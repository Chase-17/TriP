# Shared Components / Общие компоненты

Переиспользуемые компоненты без специфической бизнес-логики.

## UserAvatar.vue
Аватар пользователя.

**Props:**
- `src` - URL изображения
- `name` - имя для fallback
- `size` - sm | md | lg | xl

**Fallback логика:**
1. Показать изображение если `src` валидный
2. Показать инициалы из `name`
3. Показать дефолтную иконку

**Использование:**
```vue
<UserAvatar 
  :src="user.avatar" 
  :name="user.nickname" 
  size="md"
/>
```

## PlayerIcon.vue
Иконка игрока для списков и панелей.

**Props:**
- `player` - объект игрока
- `online` - статус подключения
- `active` - выделение (текущий ход)

**Отображает:**
- Аватар игрока
- Статус онлайн (зелёная точка)
- Индикатор активного хода

## HealthDisplay.vue
Отображение здоровья персонажа.

**Props:**
- `current` - текущее HP
- `max` - максимальное HP
- `wounds` - объект ран
- `compact` - компактный режим

**Раны (wounds):**
```javascript
{
  scratch: 2,   // Царапины
  light: 1,     // Лёгкие
  medium: 0,    // Средние
  heavy: 0,     // Тяжёлые
  critical: 0   // Критические
}
```

**Визуализация:**
- Полоска HP с цветовой индикацией
- Иконки ран по типу
- Анимация при изменении

## Паттерны

Все shared-компоненты:
- Не зависят от stores
- Получают данные через props
- Эмитят события наверх
- Минимальная внутренняя логика
