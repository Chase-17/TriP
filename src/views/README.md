# Views / Страницы

Vue Router страницы приложения.

## PlayerLobby.vue
**Страница входа для игроков.**

**URL:** `/`

**Функционал:**
- Ввод кода комнаты
- Настройка профиля (имя, аватар)
- Присоединение к игре

**Редиректы:**
- После присоединения → `/room/:roomId`

## PlayerRoom.vue (~650 строк)
**Игровая комната для игрока.**

**URL:** `/room/:roomId`

**Функционал:**
- Подключение к мастеру через PeerJS
- Отображение GameLayout
- Создание персонажа (CharacterWizard)
- Получение обновлений от мастера

**Состояния:**
1. Подключение к комнате
2. Настройка профиля (PlayerProfileSetup)
3. Создание персонажа (если нет)
4. Игра (GameLayout)

**P2P события:**
- Получение обновлений карты
- Синхронизация персонажей
- Получение результатов бросков

## MasterPanel.vue (~300 строк)
**Панель управления мастера.**

**URL:** `/master`

**Функционал:**
- Создание комнаты
- Управление картами
- Настройки игры
- История комнат

## MasterRoom.vue
**Игровая комната мастера.**

**URL:** `/master/:roomId`

**Функционал:**
- Хостинг P2P комнаты
- GameLayout с isMaster=true
- Управление всеми игроками
- Инструменты мастера

**P2P события:**
- Отправка обновлений карты
- Синхронизация персонажей
- Публикация бросков

## Маршрутизация

```javascript
// router/index.js
const routes = [
  { path: '/', component: PlayerLobby },
  { path: '/room/:roomId', component: PlayerRoom },
  { path: '/master', component: MasterPanel },
  { path: '/master/:roomId', component: MasterRoom }
]
```
